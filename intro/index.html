    AnkaFlow¶

    Write Once, Run Anywhere ‚Äî SQL-powered, YAML-defined data pipelines that work in Python or the Browser.

    Run data workflows that combine Parquet, REST APIs, and SQL transforms with no infrastructure, using DuckDB under the hood. Supports local Python, Pyodide (browser), and Chrome/Firefox Extensions.

    üöÄ Features¶

    • ‚úÖ Run pipelines in Python or in-browser (Pyodide/WebAssembly)
    • ‚úÖ Supports Parquet, REST, GraphQL, DeltaLake, BigQuery, ClickHouse, and more
    • ‚úÖ No-code pipelines via YAML: source ‚Üí transform ‚Üí sink
    • ‚úÖ Modular: browser/server backends, cloud optional
    • ‚úÖ Built on DuckDB with SQL and optional Python transforms
    • ‚úÖ Compatible with Pyodide, Chrome Extensions, and GitHub Pages

    üì¶ Installation¶

    Install only the core (minimal setup with Parquet, JSON S3 support for Pyodide or remote embedding; does not include databases):

    uv pip install ankaflow

    Install with full server capabilities (BigQuery, ClickHouse, Delta write):

    uv pip install ankaflow[server]

    Install for development:

    uv pip install -e .[dev,server]
